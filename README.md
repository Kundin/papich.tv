# React Apollo SSR

## Features

- React 16
- Babel 7
- Webpack 4
- React Router 5
- Apollo 3 with hooks
- React Helmet
- Loadable component
- [?] SSR hot reload with HMR webpack

## Installation

- `git clone https://github.com/Kundin/react-apollo-ssr.git`
- cd react-apollo-ssr
- yarn
- yarn dev
- visit `http://localhost:3000`
